## ESP code for the Homekit enabled weather station with ESP8266

This code needs to be flashed to your ESP. It will collect data from the temperature and rain sensor every 30 minutes
and send them to your Influx instance
The ESP is going into deep sleep for the time when it is not actively reading any values to save power.

To learn more check out my [Blog post at Medium](https://orangecoding.medium.com/building-a-homekit-enabled-weather-station-using-homebridge-and-an-esp8266-87aceaa4b468).
